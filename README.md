# SENTER ASN — BKPSDM Kabupaten Tana Toraja

> **Sistem Evaluasi Disiplin Kerja & Rekapitulasi Otomatis ASN**  
> Badan Kepegawaian dan Pengembangan Sumber Daya Manusia (BKPSDM) Kabupaten Tana Toraja.

---

## 📌 Fitur Utama

- 📊 **Pengolahan Presensi Otomatis**: Mengolah berkas mentah Excel presensi per OPD tanpa perlu membuat rumus satu per satu.
- 🛡️ **Penimpaan Data Bersih (Clean Overwrite)**: Menjamin data presensi OPD terisolasi dan tidak terjadi duplikasi saat re-upload.
- 📋 **Panel Kontrol Checklist 26 OPD**: Memantau status pengunggahan berkas 26 OPD utama secara *real-time*.
- 🏆 **Papan Peringkat Kedisiplinan**: Perangkingan otomatis 26 OPD Kabupaten berdasarkan 4 indikator disiplin kerja.
- 📄 **Laporan PDF Eksekutif**: Generate laporan rekapitulasi resmi siap cetak untuk pimpinan.

---

## 🛠️ Teknologi & Arsitektur

- **Frontend**: Next.js 15, React, TailwindCSS, Lucide Icons, ECharts
- **Backend**: FastAPI (Python 3.13), SQLAlchemy Async, Pydantic v2
- **Database**: Supabase PostgreSQL 15
